# Jogo da Memória — Guebly Games

> Jogo de memória com tema espacial. Encontre os pares de planetas!

## Jogar

Abra `index.html` no navegador. Não precisa de instalação.

## Como jogar

1. Digite seu nome na tela inicial
2. Clique em PLAY para começar
3. Clique nas cartas para virá-las e encontrar os pares
4. Encontre todos os pares para vencer!

## Planetas

O jogo apresenta planetas do sistema solar: Sol, Mercúrio, Vênus, Terra, Marte, Júpiter, Saturno, Urano, Netuno, Plutão, Lua e um Foguete.

## Tecnologias

- HTML5
- CSS3 (animações, transições, flip 3D)
- JavaScript vanilla
- Fonte customizada (Games Regular)
