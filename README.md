# Memory Game — Guebly Games

> A space-themed memory card game. Match the planets to win!

## Play

Open `index.html` in your browser. No build step required.

## How to play

1. Enter your name on the start screen
2. Click PLAY to begin
3. Click cards to flip them and find matching planet pairs
4. Match all pairs to win!

## Planets

The game features planets from our solar system: Sun, Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune, Pluto, Moon, and a Rocket.

## Technologies

- HTML5
- CSS3 (animations, transitions, 3D card flip)
- Vanilla JavaScript
- Custom font (Games Regular)
